App/site feito para automatizar um serviço que antes era demorado. Quando havia a necessidade de postar as cargas disponíveis, o cliente perdia horas editando uma foto.
Com esse modelo, basta alguns cliques e a foto desejada já está pronta, otimizando o resultado e, sendo assim, o cliente consegue postar mais fotos diariamente do que antes.
